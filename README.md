# Search-Algorithms

## Overview
* A desktop application implemented using **JavaFX** that contains implementation for common search algorithms including:
    * Depth-First Search Algorithm.
    * Breadth-First Search.
    * Gready Search Algorithm.
    * A* Search Algorithm.
